# Préparation Summer

Cinq cours condensés et interactifs pour réviser : Probabilités &
Statistiques, Algèbre linéaire, Analyse, SQL & Bases de données, et
Python pour la data.

Chaque matière : explications « comme à un enfant », fiches, QCM cliquables
avec correction immédiate, exercices corrigés, une section *Approfondissements*
et un examen blanc de 30 questions.

## Utilisation
Ouvre `index.html` (le portail). Tout est en HTML/JS statique — aucune
installation. La progression est sauvegardée dans le navigateur, avec :
- **Export / Import** d'un fichier de sauvegarde (pour ranger dans un Drive).
- **Synchro automatique Firebase** (optionnelle) — voir `firebase-sync.js`.

## Synchro automatique
Sur la page d'accueil, entre un **code perso** identique sur chaque appareil
pour synchroniser ta progression en temps réel. La configuration Firebase se
trouve dans `firebase-sync.js`.

## Déploiement (GitHub Pages)
Repo → Settings → Pages → Branch `main` / dossier `/root`. Le site est servi
à `https://<utilisateur>.github.io/<repo>/`.
