Les passerelles à chercher

  • Proba × Analyse : une espérance est une intégrale — ∫₀^∞ x e⁻ˣ dx = 1 = E[X] de l'Exponentielle.
  • Algèbre × Analyse : la Hessienne est une forme quadratique ; « définie positive » ⟺ minimum.
  • Algèbre × Python : np.linalg.eig rejoue tes diagonalisations à la main.
  • SQL × Python : tout GROUP BY / JOIN se refait en groupby / merge Pandas.
  • Le grand pont : la régression β̂ = (XᵀX)⁻¹Xᵀy = projection (Algèbre) = minimum d'un coût (Analyse) = code (Python) sur des données bruitées (Proba).
Synchro automatique (Firebase)
Une fois activée, ta progression se met à jour toute seule sur tous tes appareils : entre le même code perso partout et c'est synchronisé en temps réel.
Sauvegarde manuelle (fichier / Drive)
Sans Firebase, ou en plus : exporte un fichier, range-le dans ton Drive perso, puis importe-le sur l'autre appareil.

Le fichier progression-dsba.json contient tes cases cochées et tes scores QCM des 5 matières. Garde-le dans Google Drive / iCloud et réimporte-le quand tu changes d'appareil.
